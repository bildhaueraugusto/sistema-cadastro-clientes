# Sistema de Cadastro de Clientes

Sistema desenvolvido em Java para cadastro e gerenciamento de clientes, criado como trabalho acadêmico no curso de Engenharia de Software.

## Funcionalidades

- Cadastro de clientes com nome e CPF
- Listagem de clientes cadastrados
- Armazenamento em arquivo

## Tecnologias

- Java
- IntelliJ IDEA

## Como executar

1. Clone o repositório
2. Abra no IntelliJ IDEA
3. Execute a classe `CadastroClientes.java`
